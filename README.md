# Painel Financeiro

Dashboard com cotações e indicadores financeiros do Brasil em tempo real, usando a API pública do Banco Central.

## O que mostra

- Cotação do Dólar e Euro com variação do dia
- Taxa SELIC e IPCA atuais
- Gráfico histórico dos últimos 30 dias
- Tabela comparativa USD × EUR

## Como rodar
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy grátis

Suba no [Streamlit Cloud](https://streamlit.io/cloud) conectando o repositório GitHub — fica online sem custo.

## Fonte dos dados

API pública do Banco Central do Brasil · https://olinda.bcb.gov.br
